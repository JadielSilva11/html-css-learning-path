Código fonte / Pré-formatação

É possível formatar um texto escrito em linguagem de programação com a tag CODE.

Por exemplo: O comando system.out.println("Teste") é escrito em Java.

A TAG code sozinha não identa o texto, apenas formata ele de uma maneira diferente.

Para códigos maiores é possível identar o texto envelopando a tag CODE com a tag PRE. Por exemplo:

        
    #include <stdio.h>
    
    int main(){
        int num = 10;
        
        if(num % 2 == 0){
            printf("O numero %d é par!", num);
        }else{
            printf("O numero %d é ímpar!", num);
        }

        return 0;
    }

O código acima está escrito em linguagem C e verifica se um número é ímpar ou par.

Citações

Podemos destacar uma citação sem quebra de linha com a tag Q.

Para citações maiores é utilizado a tag BLOCKQUOTE que pode ser combinada com o parâmetro CITE onde pode ser inserido o link de onde a citação foi retirada, o link não aparece visualmente ao usuário, serve apenas para o navegador reconhecer de onde é aquela citação.

Exemplo: Segundo Manuel Castells (1942), sociólogo espanhol:

A internet é muito mais que uma tecnologia. É um meio de comunicação, de interação e de organização social.

Abreviações

Podemos envelopar uma sigla HTML, CSS para que o usuário saiba o significado dela passando o mouse por cima.
