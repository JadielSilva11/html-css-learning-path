Texto marcado

É possível deixar um termo marcado com a tag MARK.

A tag MARK é por padrão amarela, mas utilizando o parâmetro style do CSS é possível alterar a cor da marcação.

Este é um termo deletado com a tag DEL.

Texto sobrescrito

Pode ser utilizado com a tag SUP. ex: 102+5

Texto subscrito

Pode ser utilizado com a tag SUB. ex: H2O
