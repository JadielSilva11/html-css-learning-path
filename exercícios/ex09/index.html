Imagens dinâmicas

É possível adaptar uma imagem dependendo do tamanho da tela do usuário, para isso é necessário ter a mesma imagem com tamanhos diferentes: pequeno, médio e grande.

As Tags utilizadas para a criação de imagens dinâmicas são:
A tag img com a imagem original (imagem grande) e a tag source:media:type dentro da tag picture.

A tag source:media:type serve para dizer até qual tamanho a imagem original será carregada e qual imagem será carregada depois dela.

OBS: A tag source deve ser escrita em ordem. Por exemplo: a source que carrega uma imagem pequena deve estar acima da source de uma imagem maior!

Para verificar a mudança de tamanho da imagem, diminua o tamanho do site ou abra em outros dispositivos!
