Listas ordenadas

É possível criar listas ordenadas com a tag OL (Ordered list)!

Cada item de uma lista é criado com a tag LI (List item)

Exemplo de lista ordenada:

  1. Acordar
  2. lavar o rosto
  3. tomar café da manhã
  4. Escovar os dentes

É possível personalizar a lista com o parâmetro TYPE dentro da tag OL, para alterar a forma em que a lista será ordenada, por números, letras ou símbolos. Dentro de TYPE é permitido os parâmetros 1, A, a, I, i.

Listas não ordenadas

É possível criar listas não ordenadas com a tag UL (unordered list)!

Exemplo de lista não ordenada:

  • Pão
  • Leite
  • Ovo
  • Café
  • Manteiga

Também é possível personalizar a lista com o parâmetro TYPE dentro da tag UL com os valores "Disc", "Circle" ou "Square".

Listas de definição

É possível criar listas de definições, são como dicionários, onde possuem os campos "termo" e "definição"!

Uma lista de definição é criada com a tag DL (Definition list) e seus campos "termo" com a tag DT e "definição" com a tag DD.

Exemplo de lista de definição:

HTML
Linguagem de marcação que define a estrutura de um site.
CSS
Lisnguagem de marcação de estiliza um site.
JavaScript
Linguagem de programação que cria a interatividade de um site.

Junção de listas

É possível mesclar listas ordenadas com não ordenadas simplesmente colocando uma tag dentro da outra.

Exemplos:

Lista de tarefas

  1. Manhã
    1. Acordar
    2. Tomar café
    3. Tomar banho
    4. Escovar os dentes
  2. Tarde
    1. Almoçar
    2. Escovar os dentes
    3. Estudar
    4. Treinar
