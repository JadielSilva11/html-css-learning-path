Links externos

É possível criar links externos utilizando a tag A!

Por exemplo: Para acessar o meu github, clique aqui!

Porém, o link acima sobrescreve o site atual. Para abrir o link em outra aba é necessário utilizar os parâmetros: target="_blank" e rel="external".

Por exemplo: Para acessar o meu linkedin, clique aqui!

Para links de patrocínio ou empresas/matérias que você não tem relação ou não garante que é "confiável", é importante utilizar dentro da tag A o parâmetrio rel="nofollow"

Por exemplo: Para entrar no site do mercado livre, clique aqui!

Links internos

Links internos são utilizados para navegar para páginas dentro do próprio site.

Para criar links internos utiliza-se a mesma tag A, mas no lugar de um link para um site, basta colocar o caminho para a página.

Para navegar para a segunda página deste site, clique aqui!

Links para downloads

É possível criar links para downloads utilizando a mesma tag A junto dos parâmetros download="nome_do_arquivo" e type="tipo_do_arquivo".

Para descobrir como declarar o tipo do arquivo, acesse: iana.org
