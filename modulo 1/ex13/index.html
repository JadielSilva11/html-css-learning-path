Estilos internos

São estilos CSS feitos na mesma página do HTML, mas não na mesma linha como no inline.

É feito com a tag <style> dentro da tag cabeçalho <head> do código HTML.

Não é a forma mais utilizada, mas é uma forma útil quando se trabalha com poucos documentos e também deixa o código mais organizado pois separa o código de estilo do código de significado
