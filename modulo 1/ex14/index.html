Estilos externos

São estilos CSS feitos em arquivos específicos para estilo.

É feito um novo arquivo style.css.

É a forma mais utilizada, pois basta chamar um link para o arquivo style <link:css> no head do arquivo HTML para vincular o estilo com o código do significado.

Também é permitido misturar os tipos de implementação: CSS interno, externo e inline.
