Estilos inline

Estilos inline são tags CSS colocadas exatamente dentro da tag que você quer modificar.

Só precisa do parâmetro style="tipo_do_estilo" dentro da tag que você quer modificar.

É a maneira menos utilizada, pois é o mais trabalhoso de implementar e realizar mudanças, além de misturar os códigos de significado(HTML) e estilo(CSS).
